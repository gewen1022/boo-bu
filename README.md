# 博杯掷筊

一个基于网页的传统掷筊（博杯）工具，模拟闽南民俗中向神明请示问卜的方式。

## 在线访问

👉 **https://gewen1022.github.io/boo-bu/**

手机端和电脑端均可使用，点击「掷筊」按钮或按空格键即可掷杯。

## 掷筊规矩

### 基本用法
心中默念所求之事，掷出筊杯，以结果请示神明。

| 结果 | 形态 | 含义 |
|------|------|------|
| 圣杯 ✓ | 一凸一平 | 神明同意，所求应允 |
| 笑杯 ☺ | 两面平 | 神明在笑，可再请示 |
| 阴杯 ✕ | 两面凸 | 神明不允，另作打算 |

### 传统规矩
- 同一问题最多掷 **3 次**，3 次内无圣杯应另作打算
- 第一次即为神明回答，不宜「硬求」到圣杯为止
- 笑杯表示问题不清或神明在笑，可重新掷问
- 连续三次圣杯为「三连圣杯」，主大吉之兆

## 功能
- 🎲 点击按钮或按空格键掷筊
- 🏆 连续圣杯计数器，三连圣杯触发「大吉」庆祝动画
- 📱 响应式设计，手机端/电脑端均可使用
- 🎨 传统中国风视觉设计

## 本地运行

直接在浏览器中打开 `博杯掷筊.html` 即可使用，无需任何服务器或依赖。
